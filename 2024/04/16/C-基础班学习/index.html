#Day1 2024.04.16

ctrl + k + c //注释（可多行

ctrl + k + u //取消注释

ctrl + l //删除行

变量命名规范

  • Camel：多个单词组合变量，第一个单词首字母小写，其余的首字母大写。
  • Pascal：单个单子是指变量名，首字母大写。
